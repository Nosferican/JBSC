abstract: |
  This paper studies community formation in open-source software collaboration networks using a large-scale historical dataset of GitHub users and repositories. We combine Renewal-Nonbacktracking Random Walks with Louvain community detection to better identify small collaboration teams and characterize factors associated with community formation, including geography and programming language specialization.
